# Week 4 Module - C Programs

## Easy Level

1. **Count Digits** - counts total number of digits in a given number
2. **Reverse Number** - reverses the digits of a number
3. **Sum of Digits** - adds up all the digits of a number
4. **Even Odd Digit Count** - counts how many even and odd digits are in a number
5. **Largest Digit** - finds the biggest digit in a number

## Medium Level

6. **Prime Count in Range** - counts how many prime numbers exist between L and R
7. **Sum of Proper Divisors** - adds all divisors of a number except the number itself
8. **Print All Factors** - prints every factor of a number in order
